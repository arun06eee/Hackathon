Challenge Statement

Concept Title : Holiday Resort Reservation system

1. The Participant has to create a Holiday Resort Reservation System for "ABC Resorts". The basic features should include
  • Listing of the rooms in resort
  • Booking of rooms by Customer.
  • Admin should be able to view the list of reservations made.
  • Include graphs to show reservation reports on a selected date range.

2. The Participant can add more functionalities to showcase their technical skills.

3. Android and iOS paricipants donot have to write webservice to store data in external database. They can just store the data in sqllite or JSON file inside their mobile app.

4. All Participants has to provide the Create table scripts that they have used.
